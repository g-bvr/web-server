WebserverEndPointCommand

Purpose

Define an endpoint (endpoints are executed concurrently, the request returns after processing is done, endpoints can have url parameters that are passed as variables to the script)

Variants

UsageDescription
SERVER ENDPOINT path scriptexecute script in current workspace
SERVER ENDPOINT path script IN folderexecute script in sub-workspace specified by given folder

Arguments

ArgumentValue
pathThe url path that shall be mapped to this endpoint (starts with an / character)
scriptThe path to the script (relative to current workspace, not the execution workspace) that gets executed when endpoint receives a GET request
folderThe path to a folder (relative to current workspace) that serves as execution workspace (the executed script can be located outside this folder)

Plugin

This command is defined in WebserverPlugin
